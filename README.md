 - Login,signup,forgort password

# Usage, Installation

```
$ git clone https://github.com/Ravindu1015/project-group-01.git
$ cd jobportal
```

## Frontend

`React, Material-UI`

```
 $ cd frontend
```

```
 $ npm install
 $ npm start
```
or

```
$ yarn install
$ yarn start
```

## Backend

`Node.JS, Express.JS, Mongoose, Passport, multer, uuid, cors`

```
 $ cd backend
```

```
 $ npm install
 $ npm start
```
or

```
$ yarn install
$ yarn start
```
